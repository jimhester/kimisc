Convert Numeric to Factor, with custom formatting

Usage

cut_format(x, breaks, include.lowest = FALSE, right = TRUE, ordered_result = FALSE, ..., format_fun = format, sep = ", ")

Arguments

x
a numeric vector which is to be converted to a factor by cutting.
breaks
either a numeric vector of two or more unique cut points or a single number (greater than or equal to 2) giving the number of intervals into which x is to be cut.
include.lowest
logical, indicating if an ‘x[i]’ equal to the lowest (or highest, for right = FALSE) ‘breaks’ value should be included.
right
logical, indicating if the intervals should be closed on the right (and open on the left) or vice versa.
ordered_result
logical: should the result be an ordered factor?
...
Passed to cut
format_fun
[function(x): character] A vectorized function that performs the desired formatting. Default: format
sep
[character(1)] The separator between lower and upper end of the interval. Default: ", "

Description

This is an enhanced version of cut that allows a custom formatting to be applied to the values.
